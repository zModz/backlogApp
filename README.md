## FAQ
### **Why this app exists?**  
It came from the lack of proper tools to categorize and mark games as played or completed, other tools either didn't work properly or didn't have other features. So I pulled a Thanos and "did it myself".

### **Why there's 2 branches?**  
The `main` branch is the original project that had the majority of the screens done, but I wanted to create a quick version of the app that just had the basic functions that mostly works offline. 
Later, both versions will merge.

### **What are you using to get the games' info**  
I'm using [IGDB](https://www.igdb.com/) API.  
Later I plan on integrating [HowLongToBeat](https://howlongtobeat.com/).
